# ECH 개발 로드맵

이 문서는 ECH 개발 진행의 기준 로드맵입니다.  
앞으로 작업은 본 문서를 기준으로 수행하며, 완료된 항목은 취소선(`~~항목~~`)으로 처리합니다.

## 진행 규칙
- 새 작업 시작 전, 관련 로드맵 항목 존재 여부를 먼저 확인합니다.
- 완료된 작업은 삭제하지 않고 취소선으로 표시합니다.
- 세부 작업이 생기면 하위 항목으로 추가합니다.
- 로드맵 상태가 변경되면 `.cursor/rules/CHANGELOG.md`에 함께 기록합니다.
- 기능 개발/수정 완료 시 `docs/FEATURE_SPEC.md`와 `docs/HANDOVER.md`를 함께 업데이트합니다.
