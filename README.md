# CSTalk

> **사내용 실시간 협업 허브** — 대화·파일·업무·관리를 **한곳**에서.  
> Slack·Microsoft Teams·Flow에서 익숙한 **채널 중심 UX**를 지향합니다.

| | |
| :--- | :--- |
| **배포 기준 버전** | **1.2.11** · Git 태그 `v1.2.11` |
| **변경 요약** | `.cursor/rules/CHANGELOG.md` |

**Java**와 **Node.js**를 함께 쓰는 구조로, **안정적인 업무 처리**와 **끊김 없는 실시간 알림**을 동시에 노립니다. 브라우저만으로도 쓸 수 있고, **Windows용 Electron 앱**으로 알림·자동 업데이트까지 지원합니다.

---

## ✨ 주요 기능

### 💬 실시간 대화 · 채널

- **공개·비공개 채널**과 **1:1 DM**, **나에게 쓰기**(본인 전용 DM·메모 — 목록·헤더 표시는 일반 DM과 같이 **본인 이름**(·프레즌스)) — 프로젝트·부서 단위로 나눠 대화
- **스레드·답글**으로 맥락 유지, **스레드 모아보기**로 활발한 원글 한눈에
- **`@멘션`** — 채널·DM 멤버 기준 자동완성, 멘션 알림·**미확인 멘션 목록**·인앱 토스트/목록에서 **업무·칸반** 원클릭(메시지에서 업무 생성 후 칸반 첫 컬럼 배치)
- **읽음 포인터·미읽음 배지**, **최근 대화 순 정렬**, **퀵 레일**(미읽음·즐겨찾기 느낌의 빠른 접근)
- **온라인 / 자리비움 / 오프라인** 등 **프레즌스** — 채팅·목록·DM 헤더에 표시
- **프로필 사진** — 채팅·멤버·검색·조직도 아바타에 반영; 본인 변경은 좌측 하단 프로필 진입 시, 관리자는 사용자 편집에서 업로드(스토리지 `user-profiles/` 단일 폴더)
- **다크·라이트 테마** — 사용자 설정을 DB에 저장해 로그인 후에도 유지

### 📎 파일 · 이미지 · 첨부

- **이미지·문서 업로드**, 채팅에 **2열 그리드** 썸네일, 클릭 시 **라이트박스**
- **드래그 앤 드롭·다중 선택·클립보드 붙여넣기**로 여러 파일 한 번에; 전송 전 미리보기에서 **항목별 제거(✕)**, 추가 선택 시 목록 **누적**
- **첨부 모아보기·이미지 모아보기·파일 허브** — 채널·DM 공통으로 정리, **자료실**(폴더·핀·메모·대화로 이동)
- **연속 첨부 묶음·ZIP 일괄 저장**, **저장 후 열기**(브라우저 / **Electron**에서 OS 기본 앱)
- **미리보기·원본** 선택 다운로드 등 용량·형식에 맞는 UX
- **대용량 다운로드** 시 진행 표시(하단 상태), **전송 취소** 시 업로드 요청 중단

### 📋 업무 항목 · 칸반 · 캘린더

- 채널별 **업무 생성·상태 관리**, 메시지에서 **업무로 바로 연결**, 업무 단위 **마감일·우선순위**(LOW/NORMAL/HIGH)
- **칸반 보드** — 컬럼·카드, 담당자, 드래그로 이동, 이력 조회
- **개인 캘린더(MVP)** — 워크플로 허브에서 **내 일정** 조회·추가, 채널·DM 맥락에서 동료에게 **일정 공유** 후 상대 **수락** 시 달력에 반영(출처 표시); 백엔드에 **일정 제안·확정**(본인 달력만)·**충돌 검사** API(Phase 6-4); **iCal 내보내기·가져오기** 및 허브에서 제안 대기 목록(Phase 6-5)
- **채널 업무 허브** 모달에서 목록·칸반·저장·삭제·복원까지 일괄
- 좌측 **워크플로우** — 선택 채널·DM에 연결된 업무·칸반을 한 모달에서 관리(미선택 시 참여 채널 자동 선택) · 상단 **워크플로우** 동일 · 채팅 멤버 패널 **워크플로우**는 **현재 채널** 기준
- 상단 **CSTalk** 클릭 시 채널 미선택 **시작 화면**으로 복귀(구 대시보드 메뉴 없음)
- 사이드바 **내 할 일** — 지연·오늘 마감·멘션 연계(원본 채팅으로 이동)·담당 칸반(워크플로 모달)으로 묶어 표시, 마감 **지연·임박** 건수 배지(미읽음과 구분)

### 🔍 검색 · 멤버 · 조직

- **통합 검색** — 메시지·파일·업무·칸반 등을 한 번에 (권한·멤버십 반영)
- **이름·이메일·사번·부서** 검색, **회사→본부→팀** 조직 트리에서 멤버 선택
- 채널·DM·구성원 추가는 **검색 + 조직도 통합 모달**, 프로필에서 **DM 바로 시작**

### 🛡️ 관리자 · 운영

- 좌측 **관리자** 레일(조직·사용자·배포·설정) — 현재 화면과 메뉴 강조가 맞춰짐
- **그룹웨어 SSO** 연동, **AD 자동 로그인**(Electron·사내 PC 전제)
- **사용자·조직 동기화**, **역할·권한(RBAC)**, **통계·차트**, **오류·감사 로그**
- **배포·버전 관리**, **보존·아카이빙** 등 운영 정책을 화면에서 다루기

### 🖥️ Windows 데스크톱 (Electron)

- **설치형 EXE**, 트레이·**OS 알림**(일반 메시지·멘션)
- **자동 업데이트** — GitHub 릴리즈 또는 **내부망** 백엔드 배포 경로
- **단일 인스턴스**, **시작 프로그램 등록**, 아이콘·설치 경로 등 **엔터프라이즈 배포** 고려

## 🧱 기술 스택 (요약)

| 영역 | 내용 |
| :--- | :--- |
| **백엔드** | Java / **Spring Boot** — REST, 인증, DB |
| **실시간** | **Node.js** + **Socket.io** (Express 아님) |
| **프론트** | **Vanilla JS** — 가볍고 빠른 UI |
